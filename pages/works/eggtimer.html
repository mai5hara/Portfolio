Egg Timer

PROJECTUI/UX Design, Developent

USE LANGUAGESketch / React / Redux / Hooks / CSS3 / styled-components

CONCEPTThis is a timer web app for cooking the boiled eggs. When I wanted to add an idea that would be useful in a particular situation, rather than an ordinary timer, I realized that in my daily life, I always check the time for boiling eggs. From that behaviour, I got the idea of ​​the Egg Timer in which the time of the boiled egg in the state I want to make was set in advance.
